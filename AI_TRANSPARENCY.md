# AI Transparency

GatekeeperV2 Battery Edition used AI assistance during development.

AI assistance was involved in parts of:

- Web UI implementation.
- AMP startup and port handling changes.
- Discord embed/banner configuration changes.
- Whitelist button workflow changes.
- Permission and user management UI changes.
- Documentation and troubleshooting text.

AI-generated or AI-assisted work is not a replacement for human review. All changes should be tested in a real AMP and Discord environment before production use.

Community feedback, bug reports, and code review are welcome and should be treated as the source of truth when behavior differs from the documentation.
