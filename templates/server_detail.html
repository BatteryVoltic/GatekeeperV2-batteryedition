These groups decide where Gatekeeper posts the current display type. If Display is set to Discord Embeds, these channels receive embeds. If Display is set to Custom Banner Images, these channels receive banner images.

Assigning a server to a group decides which servers appear together in the same Discord posting set.

Create one group per Discord listing you want. Example: Public Servers, Staff Servers, Modded Servers.

What the status colors mean
  • Online: The dedicated game server is running.
  • Starting: AMP reports the dedicated game server is starting or restarting.
  • Offline: The dedicated game server is offline.

Embed settings save automatically. Uploads still use the upload button.

Embed Media

Discord image guidance
  • Thumbnail: square image, 256x256 to 512x512 works best.
  • Embed image: 16:9 or 2:1 images work best. Keep width under 4096px.
  • Upload size: this web UI caps uploads at 8 MB to stay practical for Discord embeds.

Thumbnail Editor

Crop any image into a square thumbnail. The editor saves a 512x512 PNG and selects it as this server's embed thumbnail.
